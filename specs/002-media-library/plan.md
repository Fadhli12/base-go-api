# Implementation Plan: Media Library for Go\n\n**Branch**: `002-media-library` | **Date**: 2026-04-22 | **Spec**: `specs/002-media-library/spec.md`\n\n---\n\n## Summary\n\nA production-ready polymorphic media library enabling any domain model (News, User, Invoice, etc.) to associate and manage related assets (images, documents, files). Features include file storage abstraction (S3/local), auto-generated thumbnails, permission-aware access (RBAC), audit logging, and query optimization via eager-loading.\n\n**Key Decisions** (from research.md):\n- Image processing: `govips/v2` (fast) + stdlib fallback\n- Async conversions: Hybrid (sync <5MB, Redis queue >5MB)\n- CDN: Generic signed URLs (HMAC-SHA256)\n- Cleanup: Auto TTL (30 days) + manual CLI command\n- Versioning: v2 feature (not v1)\n- Quotas: Optional, role-based, disabled by default\n\n---\n\n## Technical Context\n\n**Language/Version**: Go 1.22+  \n**Primary Dependencies**:\n- `gorm.io/gorm` (already in project)\n- `github.com/davidbyttow/govips/v2` (image processing)\n- `github.com/aws/aws-sdk-go-v2` (S3, optional)\n- `go-redis/redis/v8` (already in project, for job queue)\n- `echo` v4 (already in project)\n\n**Storage**: PostgreSQL + file backend (local or S3)  \n**Testing**: Go stdlib testing + testify + testcontainers (already in project)  \n**Target Platform**: Linux/macOS servers, containerized deployment  \n**Performance Goals**:\n- Upload: <500ms for files <10MB (includes thumbnail generation)\n- Download: <100ms for cached files\n- Thumbnails: Auto-generate within 1s of upload\n- Concurrent uploads: 1000+ concurrent without data loss\n\n**Constraints**:\n- Max single upload: 100MB (v1)\n- Soft delete mandatory (per constitution)\n- Permission checks on all operations\n- Audit logging for all actions\n- No GORM AutoMigrate (SQL migrations only)\n\n**Scale/Scope**:\n- Support 100k+ media items\n- Polymorphic to unlimited entity types\n- Multi-instance consistency (Redis pub/sub)\n- Orphaned cleanup job (daily)\n\n---\n\n## Constitution Check\n\n**Gate: Must pass before Phase 0 research.
